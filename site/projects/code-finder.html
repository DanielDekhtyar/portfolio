Project Case Study

Code Finder

An AI-assisted search companion that makes developer onboarding effortless by surfacing the right code snippet, design pattern, or API usage in seconds.

April 2024 â€“ May 2025 Role: Full-stack Engineer Stack: Python, Flask, PostgreSQL, OpenAI API

Overview

Code Finder ingests internal Git repositories, indexes symbols, docstrings, commit history, and architectural notes, and exposes a natural-language interface that ranks matches by relevance. It eliminates tribal knowledge silos and accelerates feature delivery.

Highlights

  • Designed a hybrid vector + keyword ranking engine combining OpenAI embeddings with BM25 for precise contextual matches.
  • Implemented repository sync workers using Celery and PostgreSQL LISTEN/NOTIFY to stream updates in near real-time.
  • Crafted an intuitive UI that groups matches by repository, highlights line numbers, and previews surrounding context.

Impact

Reduced onboarding time by 35% for new engineers, enabled cross-team knowledge sharing, and established a knowledge base that ties code references directly to product documentation.
